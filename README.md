# 🧠 Proyecto de Listas Enlazadas en C#

Este proyecto resuelve dos ejercicios utilizando **listas enlazadas simples** en el lenguaje de programación **C#**, como parte de una tarea académica.

## 📚 Descripción de los ejercicios

### ✅ Ejercicio 1: Invertir una lista enlazada

Implementar un método que permita invertir los datos almacenados en una lista enlazada.  
Ejemplo:  
Antes → `10 -> 20 -> 30 -> null`  
Después → `30 -> 20 -> 10 -> null`

### ✅ Ejercicio 2: Buscar cuántas veces se encuentra un dato

Crear un método que reciba un valor como parámetro y retorne la cantidad de veces que ese valor se encuentra en la lista.  
Si el valor no se encuentra, debe mostrar un mensaje indicándolo.
